Você, iniciante ou não, já se pegou procurando tutorial ou exemplos de como utilizar um sensor que você comprou recentemente? É uma coisa normal, certo? Mas hoje, iremos falar sobre um serviço muito legal que está disponível gratuitamente pela Internet.

A plataforma é uma plataforma grátis, que lhe ajuda a desenhar seu circuito. Mas não para por aqui... a plataforma também gera um código de exemplo para testar um certo componente. E agora, iremos ver um pouco mais sobre essa belezura!

Circuito.io

Corra já lá pra plataforma, através desse link. Lá na header da página, clique em App e vá direto pro workbench da plataforma. Ao abrir o aplicativo, é possível visualizar um Arduino UNO já no circuito.

Arduino UNO já vem configurado no circuito, é possível trocar por outro microcontrolador.

Ao arrastar um sensor DHT22 para o circuito, a própria plataforma já monta o circuito com o sensor conectado ao Arduino UNO. Para remover, trocar o componente por outro (utilizando a opção SWAP) ou duplicar, clique no mesmo e seleciona a opção que deseja pelo menu. No mesmo menu, é possível ler as informações a respeita do componente que você está utilzando (importante para entender o componente).

Menu do componente. É possível remover, duplicar ou ler as informações do componente por esse menu.

O código é gerado automaticamente

Ao finalizar o circuito, ou não, é possível verificar o código que a plataforma gera. Basta ir na aba Code> e visualizar o código. A plataforma cria um arquivo zipado, contendo o código que você vai enviar pro microcontrolador, nesse caso o código é feito no Arduino IDE (por isso a extensão é .ino), e as bibliotecas a serem utilizadas no circuito.

Arquivo zipado que a plataforma gera, disponível para download.

O código, em inglês, é um teste para os componentes que voê adicinou ao circuito. Provavelmente, ao selecionar um componente que tenha vários métodos de uso, talvez não te agrade mas será o suficiente para testar o componente, acredito eu. Conclusão

O Circuito.io é, de fato, uma ferramenta bastante atrativa para iniciantes. Não vejo um experiente utilizando esse sistema. O que mais me chamou atenção nessa plataforma é a possibilidade de utilizá-la para fins educacionais visto que há um set de ferramentas bem atrativas e fáceis de utilizar e gera código.
